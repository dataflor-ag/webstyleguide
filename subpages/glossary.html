Glossary

Component

A component is a collection of elements that, when combined in a tile, serve a specific purpose

Rules

  • Need to be wrapped in a MUI Paper Object
  • Should serve a specific Purpose
  • Must consist of at least two elements

Breakpoints

Breakpoints, often known as media queries is a feature of CSS 3 allowing content rendering to adapt to different conditions such as screen resolution. 

@media screen and (display-mode: fullscreen) {
    /* Code in here only applies to screens in fullscreen */
}
@media all and (orientation: landscape) {
    /* Code in here only applies in landscape orientation */
}
@media screen and (min-device-width: 500px) {
    /* Code in here only applies to screens equal or greater than 500 pixels wide */
}
