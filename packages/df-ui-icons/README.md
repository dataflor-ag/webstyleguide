# `@dataflor-ag/df-ui-icons`

> The package provides a versatile icon library for seamless integration into various user interfaces.

## Usage

```tsx
import Icon from "@dataflor-ag/df-ui-icons"

const function example() {
  return (
    <div>
      <Icon.close />
      <Icon.home />
      <Icon.mail />
    </div>
  )
}
```
