Typography - Readable and accessible text

Font Family

        html {
            font-family: "Neo Sans", sans-serif;
            font-style: normal;
        }

Register font-family at document root element (DOM).

Font Family of DATAflor is "Neo Sans", which is available in many browsers or at least most browsers do have a good "look-alike" installed. If this browser do not know an alternative to this font Fallback to a font without serifs. Do not force to use our font (do not import from a linked .otf/.ttf file) - for accessibility allow user to override font in browser settings.

Font Size - Baseline

Breakpoint "Desktop" - more than 768px

Baseline: 100%

                html {
                    font-size: 100%;
                }

Breakpoint "Tablet" - more than 480px screen width, but less than 768px

Baseline: 80%

                html {
                    font-size: 80%;
                }

Breakpoint "Mobile" - less than 480px screen width

Baseline: 60%

                html {
                    font-size: 60%;
                }

In print media designers are used to have full control about about screensize and font size. In online media we need support a wide range of screen sizes and ratios. And also allow users to zoom in/out.

Register font-size at document root element (DOM). Define all other text-based elements relative to this "default size". This allows your WebApp texts to respond to user needs (user defined zoom in/out).

In most browsers a baseline of 100% equals to 16px.

Headline (single-line headlines)

        h1 {
            font-size: 1.8rem; # +80% more than normal text
            font-weight: bold;
        }

Headline (multi-line headlines)

        h2 {
            font-size: 1.2rem; # +20% more than normal text
            font-weight: bold;
        }

Interim Headline

        h3 {
            font-size: 1rem; # same as normal text
            font-weight: bold;
            margin:0; # no empty line after headline
        }

Normal Text

        p {
            font-size: 1rem; # same as normal text
            font-weight: normal;
            line-height: 1.15rem;
            margin:0.5rem; # dense empty line after paragraph
        }

Caption Text

        p.caption {
            font-size: 0.7rem; # -30% less than normal text
            font-weight: normal;
            line-height: 1.15rem;
            margin:0.5rem; # dense empty line after paragraph
        }
